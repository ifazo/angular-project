Explore the Angular Shop

Your one-stop shop for all things Angular. This project is a simulation of an e-commerce platform built with Angular and tailwindcss.

Product management.
Display products, add products to the cart, and manage the cart. Filter products by search, page and category. Payment is simulated with the Stripe API.
User features
User authentication is implemented with Firebase. Users can sign up, log in, and log out. Dashboard access is restricted to logged in users.
